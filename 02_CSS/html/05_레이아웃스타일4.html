position

        요소 배치(위치) 관련 스타일
        위치를 지정하는 속성

        - position:relative
         : 지정된 요소 내부의 다른 요소가 상대적인 위치를 지정할 수 있도록
           기준이 되게 만드는 속성
        
           -position:absolute
         : 기본 요소 배치 순서를 무시하고, 지정된 절대 위치에 요소를 배치
           가장 우선으로 보여지는 디자인
        
           - position:fixed
         : 항상 고정된 위치에 요소를 배치함(면의 이동 여부와 상관 없음)
        
         - position:sticky
         : 원래 요소가 보여지는 화면이라면 position x
           요소가 보여지는 화면을 벗어나면 fixed(고정)

        - z-index : 단독으로도 많이 사용
                    relative, absolute, fixed, sticky 와 같이 사용됨
                    -2147483648 : 32bit 기준으로 최소값
                    -1 다른 요소 아래에 쌓음
                     0 default
                     1 다른 요소 위에 쌓임
                    10 더 높은 우선순위, 더 많은 요소 위에 보여짐
                    2147483647 : 32bit 기준 최대값
