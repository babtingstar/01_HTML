display flex - justify-content

가로 정렬(수평 중앙 정렬)

      justify-content : div 와 같은 태그를 수평 방향으로 정렬하고 간격을 조절
      
      display:flex를 작성하면 justify-content:flex-start가 기본값
      justify-content:flex-start : 왼쪽 정렬
      justify-content:flex-end : 오른쪽 정렬
      justify-content:center : 가운데 정렬
      
      태그들의 특정적인 규칙을 제공한 상태로 정렬할 때 사용하는 방법
      justify-content:space-between : 첫 요소는 왼쪽 끝
                                      마지막 요소는 오른쪽 끝
                                      나머지 요소는 균등한 간격을 유지
      |요소1        요소2        요소3        요소4|                               
              8             8           8
      justify-content:space-around : 각 요소 사이에 동일한 간격을 줌
                                     양쪽 끝에는 반 간격씩 제공
      |   요소1      요소2      요소3      요소4   |
        3        6         6         6          3
      justify-content:space-evenly : 요소 각 사이와 양 끝에 동일한 간격을 제공
      |     요소1     요소2     요소3     요소4     |
         5         5         5        5         5

display:flex + justify-content:center

display:flex = 세로로 된 div를 가로로 보기

justify-content:center = 가로에 있는 div를 수평 중앙 정렬
