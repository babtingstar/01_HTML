요소의 영역(여백) 관련 스타일

        HTML 요소는 총 4가지의 영역으로 구성되어 있음
        
        요소 ≒ 태그
         - 태그(tag) : 
 알파벳을 열고 닫은 형식
         - 요소(element) : 
안녕하세요
 내용의 안녕하세요 를 포함함
        
        1. content(내용)의 영역
            - 요소의 내용이 작성 되는 영역(시작태그 ~ 종료태그 사이에 작성되는 내용)
        2. padding
            - content 와 border 사이의 공간
        3. border
            - 요소의 테두리가 지정되는 영역
            - content 보다 바깥쪽에서 content를 감싸고 있음
        4. margin
            - 다른 요소와의 간격을 나타내는 영역

      - 다른 요소와의 간격을 자동해서 가운데 정렬하게 만드는 속성 값
      - 기본적으로 좌우 간격만 자동으로 지정
      - 지정되는 대상은 block 형식의 요소
        inline, inline-block은 text-align:center; 사용

      text-align:center;
        - inline : span, a, strong, em, img 등
        - inline-block : button input img 등

      margin:auto;
          - block : div, h1~h6, p, ul, ol, li, section 등
