사전적 의미 : 배치, 정리
        기술적 의미 : 구성요소를 제한된 공간에효율적으로 배치하는 것

화면 배치(표시/형식) 지정 : display 속성

        요소가 화면에 어떻게 보여질지 형식을 지정하는 속성
        - block : 화면을 수직 분할(행 분할/ 한 줄 차지)
            width, height 지정 가능
        - inline : 화면을 수평 분할(행 분할 / 글자, 문자열 생각)
        - inline-block : inline의 수평 분할 + block의 width, height 지정 가능
        - none : 요소는 존재하나 화면에는 표시하지 않음
        - flex : 요소간의 간격 및 정렬을 유연하게 처리
        - grid : 격자무늬 형식으로 화면을 분할

block 형식의 요소를 inline 형식으로 변경
