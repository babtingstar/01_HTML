        하이퍼링크 관련 태그
        하이퍼링크 관련 태그
        
        a 태그 내부에 속성으로 href를 작성하면 글자색과 밑줄이 생김
        - 파란색 글자색 : 클릭하지 않은 링크
        - 보라색 글자색 : 클릭한 적이 있는 링크

        a 태그는 사용자가 특정 행동을 했을 경우를 3가지 경우로 나누어 설정
        - a:hover 
            주로 밑줄 추가(text-decoration : underline)
        - a:active
            주로 글자색 변경 / 글자크기 를 변경
        - a:visited (사용자가 이미 방문한 링크에 스타일을 지정할 때 사용)
            주로 color 만 지정

        button input type="submit" 과 a 태그의 차이점
        - button input type="submit" 태그는 제출을 위해 사용
        - a href 태그는 특정 주소나 링크로 이동하기 위해 사용
