Flex 레이아웃

        display: flex;
        div 안에 작성한 태그들을 유연하게 배치하는 속성
        - 수평이나 수직으로 정렬이 가능
        - flex는 div(block 형태)를 inline 형태로 변경

        ※ display: flex;와 같이, 자주 쓰이는 속성

        1. 가로로 존재하는 태그(ㅁ ㅁ ㅁ)의 여백을 설정하는 방법
        1-1. justify-content:center
            flex 컨테이너 내의 자식 태그를 가로 방향으로 가운데 정렬
        1-2 justify-content:space-around;
            flex 컨테이너 내의 자식 태그들 사이에 균등한 여백을 주어 배치하는
        
        2. 세로로 존재하는 태그 줄의 여백을 설정하는 방법
            align-items : center;
            세로 줄 기준 가운데 정렬
            주로 flex-direction:row;을 사용할 때, 세로 가운데 정렬이 필요할 경우 사용
            text -align -items
            └ 가로 ┴ 세로 ┘
        
        3. 태그의 너비가 줄을 초과하면 자동으로 줄바꿈이 되도록 설정
            flex-wrap: wrap;
            줄을 초과했을 때만 줄바꿈이 되는 것

flex 와 inline의 차이

특징 요소 배치 방식 크기 제어 정렬 옵션 응용 범위
inline 같은 줄에 나열되며 필요한 공간만 차지 높이 / 너비 지정 어려움 정렬 옵션 제한적 텍스트 / 링크 요소 등에 사용
flex 가로 / 세로 방향으로 유연하게 배치 크기 / 여백 쉽게 제어 다양한 정렬 옵션 지원 전체 레이아웃 설계 주로 사용
