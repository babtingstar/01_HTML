자바스크립트의 변수(Variable)

        변수
        - 메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간
        - 변수에 이름을 붙이고(변수명) 원하는 값을 대입하면,
          필요한 곳에서 변수명을 호출하여 저장된 값을 사용할 수 있다.
        
변수명 명명 규칙

        - 의미 있는 이름으로 작성
        - 대/소문자 구분
        - 특수문자는 _(언더스코어) 사용 가능(단, 첫 글자는 제외)
        - 숫자 사용 가능(단, 첫글자는 제외)
        - 카멜 표기법 작성
          [카멜 표기법(Camel Case)]
          문장, 여러 단어를 변수명으로 작성할 대 이어지는 첫 번째 단어를 대문자로 작성
          변수명을 작성할 때 띄어쓰기가 안됨
          ex) user name = userName
          변수명 짓는데 어려움이 있다면 : https://www.curioustore.com/

JavaScript의 변수와 특징

        변수 이름을 새로 만들어서 변수 공간을 생성
        
        NEW 바구니A = '물건이나 숫자가 담겼습니다.';
        변수 = Variable
        var 바구니A = '물건이나 숫자가 담겼다.';

        var : 변수. 변수명 중복o - 덮어쓰기나 여러가지 문제가 생김
        let : 변수. 변수명 중복x

        const : 상수. 중복x 값변경 x(제일 많이 사용)

        1순위 : const(JS는 특정 요소를 선택해서 사용하는 경우가 많아 변수에 고정해둠)
        2순위 : let
        3순위 : var

자바스크립트의 자료형

        자바스크립트는 변수 선언 시 별도의 자료형을 지정하지 않음
        → 변수에 대입되는 값(리터럴)에 의해서 자료형이 결정됨

        - 숫자는 ''/"" 없이 표기가 가능
        - 문자열 ''/"" 로 감싸서 작성(text)
        - true/false는 ''/"" 없이 표기가 가능
        - function(함수, 기능)
        - undefined(정의되지 않은 변수) : 선언은 됐으나 값이 들어가지 않음

        (참고)null : 참조 없음
        → document.getElementByID("오타") : 일치하는 요소가 없어 참조할 수 없다.
