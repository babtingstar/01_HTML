for 문

        - 특정 프로그램의 흐름을 제어하는 제어문 중 하나
          특정 코드를 반복할 때 사용하는 구문
        - 끝이 정해져 있는 반복에서 주로 사용(N회)

        [작성법]

        for([1]초기식; [2]조건식; [3]증감식)
        {
            [4]조건식이 true 일때 수행하는 코드;
        }

        [1] ~ [4] 1회 수행 이후,
        [2] ~ [4] 을 조건식이 false가 될 때 까지 반복

중첩 반복문

      - 반복문 내에 반복문이 작성된 형태
      - 메모리 사용량이 많아 사용을 지양

      [작성법]
      for(초기식1; 조건식1; 증감식1;)
      {
        초기식1이 true 일때 실행
        for(초기식2; 조건식2; 증감식2;)
      }

while 문

      - 끝이 정확하게 정해져 있지 않은 반복에서 주로 사용(ex 특정 값이 입력되기 전까지 반복)

      [작성법]

      while(조건식)
      {
        조건식이 true 일 경우 반복 수행;
      }

      - 조건식이 true 인 경우 내부 코드를 1회 반복 후 다시 조건식 검사 실행
      - false가 나올 때 까지 무한 반복
