조건문 (Condition)

        condition : 조건에 따라 다르게 동작
                    조건 = true 일때 작업을 수행
        
        프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로
        작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if 문

        
            if(조건식)
            {조건식=true 일 때 실행할 코드;}
        
    
        * 조건식 : 결과가 true, false 인 식
        * 조건문 : 조건식의 결과에 따라 실행을 제어하는 구문 (if, switch)

입력받은 수가 양수인지 아닌지 검사

if - else 문

        
            if(조건식)
            {조건식=true 일 때 실행할 코드;}
            else
            {조건식=false 일 때 실행할 코드;}
        
            if-else   VS   삼항연산자
            if-else : 수행해야 하는 구문이 많을 때 사용
            삼항연산자 : 수행해야 하는 구문이 적을 때 사용
            
            else(){} ----- X
            else{} ----- O

if - else_if - else

      
        if(조건식A)
        {A = true 일 때 실행;}
        else if(조건식 B)
        {A = false 이고, B = true 일 때 실행;}
        else
        {A = false 이고, B = false 일 때 실행;}
      

      else if는
      무조건 if{} - else{} 사이에 작성
      if{} - else{} 사이에 작성 횟수 제한 x

switch 문

switch, case, default 문

      switch 문
      - 식 하나의 결과로 많은경우의 수를 처리할 때 사용하는 조건문

      
        switch(식){ // 식의 결과는 정수 or 문자열
          case 결과1 :
          결과1 일때 수행할 코드;
          break; 결과1 일때 정지

          case 결과2 :
          결과2 일때 수행할 코드;
          break; 결과2 일때 정지
          
          case 결과3 :
          결과3 일때 수행할 코드;
          break; 결과3 일때 정지
          ...

          default :
          위 모든 case가 false 일때 수행하는 코드
        }

조건이 여러개인 switch 문

    if 문 처럼 조건이 여러 경우 일 때는
    
    case 조건1 : case 조건2 : case 조건 3:
    조건1이나 조건2나 조건3 일 때 부합할 경우 실행할 코드
    
    case 조건4 : case 조건5
    조건4나 조건5일때 부합할 경우 실행할 코드
    
    default :
    조건1~조건5까지 모두 부합하지 않을 경우 싫행할 코드
    
      switch(식){ // 식의 결과는 정수 or 문자열
        case 조건1 : case 조건2 : case 조건 3:
        조건1이나 조건2나 조건3 일 때 부합할 경우 실행할 코드
        
        case 조건4 : case 조건5
        조건4나 조건5일때 부합할 경우 실행할 코드
    
        default :
        위 모든 case가 false 일때 수행하는 코드
      }
    

    case 조건1: case 조건2: case 조건3:
    if(조건1 || 조건2 || 조건3)
