배열(Array)

        - 변수를 묶음으로 다루는 것 (자료구조의 기초)
            JS 변수의 특징 : 대입되는 값에 따라서 자료형이 변함
        
        JS 배열
            - 자료형 상관 없이 여러 값을 한 번에 다룸
            - 자료형 제한 x(대신 보통 한 가지 자료형만 저장)
            - 길이 제한 x(자동으로 배열 길이가 늘었다, 줄었다 가능)
        

        [Js 배열 선언 방법]

        1. const arr = new Array(); // 0칸짜리 비어있는 배열 생성ㅇ
        2. const arr = new Array(); // 3칸짜리 배열 생성(각 칸의 값은 비어있음)
        3. const arr = []; // 0칸짜리 배열 생성
        4. const arr = [value1, value2, value3] // 3칸짜리 배열 생성, 각 칸의 값은 value1 value2 value3

        index : 각 자리 위치
        length : 총 길이
