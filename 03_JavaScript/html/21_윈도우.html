window 내장 객체

        - 인터넷 창 자체를 나타내는 객체

        - 자바스크립트 모든 기능 중 최우선

        - windows의 자식 객체
            1) DOM (Document Object Model)
                : 브라우저에서 내용(페이지) 출력 부분

윈도우의 기능임을 생략해도 되는 기능들

        (window.)alert("글자")
            └> 생략 가능
        
        (window.)prompt("글자")
            └> 생략 가능
        
        (window.)setTimeout(함수, 지연시간)
            └> 생략 가능

앞에 윈도우를 붙여줘야 하는 기능들

        (window.)location.href = "이동할.html"
            └> 생략 가능하지만,
                상황에 따라 파일을 가져오는 위치가 변경될 수 있기 때문에
                명확한 위치를 표기하기 위해 window를 붙여 줌
            
            031 - 경기도
            033 - 강원도
            전화번호로 지역을 표기하는 것 처럼,
            각 기기의 주소도 위치를 설정할 수 있고, 위치가 잡히기도 함

            미국 = 3.xx
            한국 = 121. xxx
            중국 = 36.xxx

            window = 내 컴퓨터
            location = 내가 지금 보고 있는 페이지(html 파일)
            href = 주소를 가져와라
**href와 src의 차이
        링크나 리소를 참조할 때 사용하지만 사용 용도가 다름

        [href] : 페이지 이동이나, 기능을 참조할 때 사용
                    ex) css파일, 페이지간 이동, 외부 웹 페이지 링크 등
        [src] : 페이지 안에 이미지, 동영상, JS 파일 등 외부 소스를 삽입할 때 사용하지만
                    ex) JS파일, 이미지 동영상 등
