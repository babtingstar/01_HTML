복합 대입 연산자(+=, -=, /=, %=)

        기존 변수에 저장 되어있는 값에
        새롭게 대입되는 값의 산술 연산 결과를 대입
        (증감 연산자는 +씩 증감, 복합 대입 연산자는 원하는 만큼 증가, 감소)

        ex)
        let temp = 10;
        temp += 5; // temp = temp+5;
        결과 : temp 는 15

현재 값 : 0

+= (더하기 연산자)

      기존 값에 새로운 값을 더하거나 이어 붙여 업데이트하는 연산자는
      
      ==(동등연산자) : 타입을 알아서 수정, 수정이 안되는 경우도 있음
      += : 글자 + 글자 이어붙이기
           숫자 + 숫자 누적시켜 크기를 키울 수 있음

           const A = "안녕"
           const B = "반가워"
           A += B
           → A = "안녕 반가워"
           

           let C = 5
           let D = 5
           C += D
           → C = 10

           let E = "5"
           let F = 5
           E += F
           → E = 55(우선) or 10
