함수(function)

        - 목표하는 결과를 만들기 위한 코드의 묶음
        - 코드를{} 으로 묶어 작성하고, 붙여진 이름을 호출해서 사용

        * 선언, 정의, 호출

        [기본 함수 형태]

        function 함수명() // 함수 이름 선언
        {함수 내용} // 함수 정의

        함수명();  // 함수 호출

        함수란 ? 특정적인 기능을 동작하고자 할 때 특정 기능을 묶어서 만드는 명칭

        ex)
        function 전자레인지작동(N초){
            1. 코드를 꼽는다
            2. N초 만큼 시간을 선택한다
            3. N초 시간만큼 실행한다
            4. N초 시간이 다 되면 종료한다
        }

매개 변수(Parameter), 전달 인자(Argument)

        *매개변수(媒介變數)
            : 함수 선언 시 () 안에 작성되는 변수
                - 전달 인자로부터 받은 값을 저장
                - 함수 정의 부분에서 사용 가능


        중간 역할을 하거나 전달하는 변수

        parameter : 변경이 될 기준을 제공

        *전달인자(傳達引子)(Arguments)

        argument : 제공되는 정보, 실제 값
        함수(function)가 호추로딜 때 Argument의 값이 파라미터에 전달

무한한 결과가 아닌, 선택지가 있는 결과를 작성할 때 매개변수와 전달인자를 사용 매개변수와 전달인자는 한 세트 매개변수는 전달인자를 전달해주는 역할일 뿐 function 결정사(let 소개팅대상)

매개변수와 전달인자

return(반환)

        - 현재 수행 중인 함수를 종료하고 호출한 곳으로 돌아감
        - 호출한 곳으로 돌아가 때, return 옆에 작성된 값을 가지고 돌아감

익명 함수(이름이 없는 함수)

      [기본 함수]
      function functionA(paramA){
        //함수 정의
      }

      functionA(); 로 호출

      ============================

      [익명 함수]
      function(paramB){
        //함수 정의
      }
      
      함수명이 없어서 호출 할 수 없음
      기본 함수와 다르게 변수에 대입하거나, 매개변수로 바로 전달되는 곳에서 주로 사용
      ex)
      
      const 결과 = 더하기(1, 2); = 3
      function 더하기(a, b){return a + b;}
  
      const 결과 = function(a, b){return a+b;}

화살표 함수(arrow function)

      - 익명 함수의 한 종류
      - 익명 함수를 간단히 표기하는 방법
      
      익명 함수 : function(){}

      1. 기본 형태 : () => {}
      매개 변수가 1개면 () 생략 가능, 2개 이상은 생략 불가능 및 , 로 구분지어서 작성

      2. 매개변수가 1개 : (e) => {}
                          e  => {} // () 생략

      3. 매개변수가 N개 : (x1, x2, x3) => {}
                        // () 생략할 수 없음
